My Experience

First of all, I need to mention that I am a very basic programmer, who does project once in a blue moon. This is my first hacktoberfest and this pull request will be the second pull request in this contest.

Now moving to the part that what I have realised after participating in this contest and learning git, github and open source contribution. I feel that open source is very much useful for any kind of developer. It is a win-win situation for everyone. This great opportunity comes with the help of git and github. This hacktoberfest is best for the newbies to kickstart their journey with git and github. These tools will be necessary for all the developers in the career for sure. So, with the help of hacktoberfest they can be motivated and learn something great.

Many people are not understanding the purpose of this event. Many of these people are just running behind the incentives without doing any good for both himself as well as others. The main aim of participating this event should be learn new thing and get accustomed with what projects are in real life, what are the issues people are facing and how we can solve. That's all my from side.

All the best everyone. Just focus on learning, which will get you many more incentives in your whole career.
