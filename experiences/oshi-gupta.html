Tips for beginners :

Search for tags like "Good first issue" generally these are very easy to start with. You only have to create PR that need not to be labeled as "hacktoberfest". If you feel you can resolve an issue which is not labeled as "hacktoberfest", you can do it and it will be counted as a valid PR!
