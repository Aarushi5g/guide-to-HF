Learning about Open Source

This is my first year participating in Hacktoberfest. There is still a lot for me to learn, but I’m feeling good about this introduction to open source. Here are a couple of quick tips for open source newbies:

  1. Start learning git basics as soon as possible. Learn the most common git commands and practice them so they are familiar. Then, learn a couple more commands or concepts regularly so that you continue to build your git skills.
  2. Take a look at various GitHub repos. Doing this will help you get familiar with the ways projects are structured. Look through any branches, issues, and pull requests to get a sense of the project’s history.
  3. Start looking for open source projects early. You don’t have to wait for Hacktoberfest to begin making contributions. If you find a project or two you like, start looking through the code and read the contribution guidelines. Reach out to the repo owner or other contacts if you have questions.

I hope you have a wonderful time making open source contributions. Happy coding!
